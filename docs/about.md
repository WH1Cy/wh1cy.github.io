## 技术栈

<div class="grid cards" markdown>

-   :material-cog:{ .lg .middle } __静态站点生成器__

    ---

    [MkDocs](https://www.mkdocs.org/){ target="_blank" rel="noopener" } — 基于 Python 的文档站点生成器，配置简单、扩展丰富。

    ---

    [GitHub Pages](https://pages.github.com/){ target="_blank" rel="noopener" } — 静态站点托管；[Git](https://git-scm.com/){ target="_blank" rel="noopener" } & [GitHub](https://github.com/wh1cy/wh1cy.github.io){ target="_blank" rel="noopener" } — 版本控制与仓库托管。
