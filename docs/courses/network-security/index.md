# 网络安全原理与实践

本课程笔记与资料索引，包含实验报告与课程考核说明。

## 课程概要

- 网络安全核心概念：威胁建模、攻击面分析与分层防护。
- 常见主题：网络协议安全、常见网络攻击与防御（如嗅探、欺骗、DoS/DDoS）、入侵检测与流量分析、零信任与安全运维等。
