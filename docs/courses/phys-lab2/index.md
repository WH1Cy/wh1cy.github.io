# 普通物理学实验 II

本课程笔记与资料索引，包含所有实验报告的完整记录。

## 课程概要

普通物理学实验 II 是经典物理实验课程，按实验顺序涵盖：

- **虚拟仿真**：托卡马克等离子体物理
- **半导体物理**：PN 结特性、玻尔兹曼常数测定
- **薄膜测量**：椭偏仪测厚度与折射率、范德堡法测薄膜电阻
- **电路实验**：RLC 串联电路暂态与稳态、交流电桥
- **流体力学**：液体粘滞系数的测定

通过本课程的学习，你将掌握经典物理实验的基本原理与操作技能，培养实验设计与数据分析能力。
