# 隐私计算与数据合规

本课程笔记与资料索引，包含所有实验报告与课程项目的完整记录。

## 课程概要

- 隐私计算技术与方法：多方安全计算、联邦学习、同态加密、差分隐私等
- 数据合规与隐私保护：数据安全法、个人信息保护法、数据跨境传输等
- 常见主题：隐私保护技术原理、数据合规框架、隐私计算应用实践等
