# 计算机系统 II

本课程笔记与资料索引。

## 课程概要

- 深入理解计算机系统：程序如何在硬件上运行、操作系统与体系结构等。
- 常见主题：进程与线程、内存管理、I/O、并发与同步、系统编程等。
